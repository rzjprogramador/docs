# VER CONTAINERS EM USO
```docker ps```

# VER TODOS CONTAINERS EM USO E PARADOS
```docker ps -a```

# PARAR CONTAINER
> OBSERVACAO PARA TOMAR ALGUMA ACAO - TIPO DELETAR - TEM QUE USAR ESTE COMANDO DE STOP PARA PARAR-LO
```docker stop <CONTAINER_ID>```

# REMOVER VARIAS IMAGES: com images rmi
```docker images rmi <image_id> <image_id> ``` 

DELETAR CONTAINERS : com container rm
```docker container rm <container ID> <container ID>```
